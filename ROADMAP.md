# Roadmap

## v0.1 - Prompt-Based MVP

- Add system prompt and module prompts
- Define diagnostic feedback format
- Add IELTS learner profile
- Add vocabulary bank and review rules
- Add GitHub issue templates for future learning pain points

## v0.2 - Structured Data MVP

- Add JSON schema for learner profile
- Add error-pattern data model
- Add vocabulary record schema
- Add spaced review scheduler logic
- Add sample test cases for diagnostic rules

## v0.3 - Notion Integration

- Connect wrong vocabulary answers to a Notion database
- Add review status and next-review date handling
- Add YouGlish pronunciation links
- Add weekly vocabulary review summary

## v0.4 - Web App Prototype

- Build a simple web interface
- Add module selection: Reading, Listening, Vocabulary, Speaking, Writing
- Add practice history
- Add weekly diagnostic summary

## v0.5 - Evaluation

- Add rubric-based Writing and Speaking feedback examples
- Add before/after learner answer examples
- Add unit tests for diagnostic inference rules
- Add demo video or screenshots for portfolio use
