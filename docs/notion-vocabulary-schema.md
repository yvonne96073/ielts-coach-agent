# Notion Vocabulary Schema

Recommended database name:

`IELTS Vocabulary Error Bank`

## Properties

| Property | Type | Purpose |
|---|---|---|
| Word | Title | Target word or phrase |
| Chinese Meaning | Text | Chinese explanation |
| English Meaning | Text | Short English definition |
| Example Sentence | Text | IELTS-style sentence |
| Pronunciation Link | URL | YouGlish or dictionary pronunciation link |
| Sound Note | Text | Stress, reduction, or connected speech note |
| Collocation | Text | Common phrase pattern |
| My Wrong Answer | Text | Learner's incorrect answer |
| Correct Answer | Text | Correct answer |
| Error Type | Select | meaning, collocation, paraphrase, spelling, pronunciation, listening recognition, active use |
| Skill | Select | Reading, Listening, Writing, Speaking, Vocabulary |
| Topic | Select | IELTS topic family |
| Recognition or Active | Select | Recognition, Active, Both |
| Review Status | Select | Repeat Tomorrow, Review Soon, Mastered |
| Next Review | Date | Next review date |
| Times Wrong | Number | Number of misses |
| Times Reviewed | Number | Completed reviews |
| Source | Text | Quiz, reading passage, transcript, writing correction |

## YouGlish Link Format

```text
https://youglish.com/pronounce/WORD/english/uk
```

For phrases, replace spaces with `%20`.
