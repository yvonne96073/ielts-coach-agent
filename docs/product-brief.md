# Product Brief

## Product

IELTS Coach Agent is an adaptive learning coach for IELTS candidates who need more than generic practice questions.

## Target Learner

- Current level: around Band 6
- Target level: Band 7.5
- Main weaknesses:
  - Reading: paragraph focus and over-inference
  - Listening: map questions, distractors, and key-information tracking
  - Vocabulary: forgetting words and weak pronunciation recognition
  - Speaking: simple sentence repetition and tense drift
  - Writing: Task 1 recall, word count, spoken-style Task 2 writing, tense control

## Core Value

The agent observes how the learner answers, infers the likely method behind mistakes, and recommends targeted drills.

## Differentiator

Most IELTS tools provide questions and model answers. This project focuses on diagnosis:

- Why did the learner choose this answer?
- What strategy did they probably use?
- Which error pattern is repeated?
- What should they practise next?
