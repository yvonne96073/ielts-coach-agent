# Learning Loop

The agent follows a diagnosis-first loop.

1. Give a small task.
2. Collect the learner's answer.
3. Ask for reasoning when useful.
4. Mark correctness.
5. Infer likely method.
6. Label error pattern.
7. Teach one correction rule.
8. Assign a targeted drill.
9. Save vocabulary or repeated errors to review.

## Feedback Format

```text
Correctness:
Evidence / scoring reason:
Likely method used:
Error pattern:
Better method:
Next drill:
```

## Weekly Summary

At the end of each week, the agent should summarise:

- top recurring Reading errors
- top recurring Listening errors
- top recurring Writing errors
- top recurring Speaking errors
- vocabulary review status
- recommended practice focus for next week
