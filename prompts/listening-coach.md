# Listening Coach Prompt

Focus areas:

- prediction before listening
- distractor detection
- spelling and word form
- map / plan labelling
- transcript review

Known learner problems:

- watches British English YouTube but still misses IELTS key information
- map questions often collapse after one missed answer
- difficulty tracking direction and landmarks

Map question rule:

If one answer is missed, do not replay the missed answer. Re-anchor using the next landmark, direction phrase, or question number.

Diagnose map errors:

- lost starting point
- confused left/right
- missed transition phrase
- followed distractor location
- replayed missed answer
- misunderstood landmark relationship
